# ⛩️ Tori Studio

**A Experiência Imersiva de Webcomics e AudioStorys**

Tori Studio é uma plataforma de streaming premium, inspirada na estética minimalista e fluida da Netflix, projetada especificamente para amantes de quadrinhos digitais e narrativas imersivas.

## ✨ Principais Funcionalidades

### 🎧 AudioStory Player Imersivo
Uma nova forma de consumir histórias. O player estilo **Visual Novel** oferece:
-   **Cenas Cinematográficas**: Ilustrações em alta definição que preenchem a tela.
-   **Narração Sincronizada**: Legendas dinâmicas e controles de áudio completos.
-   **Navegação Integrada**: Passe as cenas manualmente ou deixe no modo automático.
-   **Ficha Técnica**: Veja detalhes sobre autores e o elenco de dubladores.

### 📖 Leitor de Webcomics Premium
-   **Scroll Infinito**: Experiência de leitura vertical fluida e rápida.
-   **Navegação de Capítulos**: Pule entre episódios sem sair do leitor.
-   **Modo Imersivo**: Interface que desaparece para focar 100% na arte.

### 👤 Experiência Personalizada
-   **Gerenciamento de Perfis**: Crie e edite perfis com avatares únicos.
-   **Minha Lista**: Salve suas obras favoritas em uma biblioteca dedicada.
-   **Exploração Inteligente**: Filtros por categorias (Ação, Fantasia, AudioStorys, etc).

## 🚀 Como Executar Localmente

**Pré-requisitos:** Node.js (v18+)

1. **Clonar e Instalar:**
   ```bash
   npm install
   ```

2. **Iniciar em Desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Gerar Build de Produção:**
   ```bash
   npm run build
   ```

## 🛠️ Tecnologias Utilizadas

-   **React 19** & **Vite 6**
-   **Tailwind CSS** (Design System Customizado)
-   **Framer Motion** (Animações Coordenadas)
-   **Lucide React** (Iconografia)
-   **TypeScript** (Segurança e Tipagem)
